# S2A-3: Top Liquidations Feed

**Epic:** Sprint 2 Expandida — Parte A (Free Tier Robusto)
**Status:** Ready for Review
**Type:** Frontend + Backend
**Effort:** S (meio dia)
**Priority:** P2
**Depends on:** S2A-0 (publisher aggregation)

## Story

Como trader visitando o CryptoVision, quero ver um feed em tempo real das maiores liquidações recentes, para sentir o pulse do mercado e identificar movimentos de cascata.

## Acceptance Criteria

### AC-1: Liquidation feed component
- [ ] Componente `TopLiquidationsFeed` exibe as últimas 50 liquidações ordenadas por valueUsd desc
- [ ] Cada item mostra: timestamp | symbol | exchange badge | side (LONG/SHORT) | quantity | valueUsd
- [ ] LONG liquidations: texto/ícone #EF4444 (vermelho, "bearish for longs")
- [ ] SHORT liquidations: texto/ícone #22C55E (verde, "bullish squeeze")
- [ ] Exchange badges coloridas: Binance #F0B90B, Bybit #F7A600, OKX #FFFFFF
- [ ] Animação de entrada: novos items deslizam do topo (150ms ease-out)
- [ ] Liquidações > $100K destacadas com fundo #F59E0B/10

### AC-2: Data source
- [ ] API endpoint: `GET /api/v1/public/liquidations/feed?limit=50`
- [ ] Lê `cv:cache:liq:agg:{symbol}:latest` para os 20 símbolos
- [ ] Ordena por valueUsd desc, retorna top N
- [ ] Delayed 15min para free tier
- [ ] Auto-refresh a cada 10s

### AC-3: Landing page integration
- [ ] Feed integrado na landing como sidebar ou seção compacta
- [ ] Mostra top 10 na landing, link "See full feed" para expandir
- [ ] Compact mode: 1 linha por liquidação, scroll vertical

### AC-4: Testes
- [ ] Teste: Feed renderiza com mock liquidations data
- [ ] Teste: Side mapping correto (LONG=vermelho, SHORT=verde)
- [ ] Teste: Large liquidations (>$100K) têm highlight
- [ ] Teste: API endpoint retorna ordenado por valueUsd desc

## Dev Notes

**UX insight:** Este feed cria "stickiness". Traders ficam olhando liquidações como quem olha ticker tape. É o equivalente visual de "algo está acontecendo no mercado agora". O valor é emocional + informacional.

**Design System:**
- Compact list: gap 4px, padding 8px per row
- JetBrains Mono para valores ($), Inter para labels
- Whale liquidation (>$100K): subtle amber glow background
