# S2B-5: Dashboard Autenticado Básico

**Epic:** Sprint 2 Expandida — Parte B (Ciclo de Venda)
**Status:** Draft
**Type:** Frontend
**Effort:** L (2 dias)
**Priority:** P1
**Depends on:** S2B-1 (auth), S2B-4 (tier gating)

## Story

Como usuário autenticado do CryptoVision, quero um dashboard com meus dados de mercado em tempo real (se pago) ou delayed (se free), para monitorar o mercado sem sair da plataforma.

## Acceptance Criteria

### AC-1: Dashboard layout
- [ ] Route `/dashboard` com sidebar navigation
- [ ] Sidebar: Overview, Alerts, API Keys, Settings
- [ ] Top bar: user email/name, plan badge, logout button
- [ ] Responsive: sidebar collapses on mobile

### AC-2: Overview page
- [ ] Default dashboard view with market summary widgets
- [ ] Reuses heatmap, funding comparison, and liquidations feed from Parte A
- [ ] Real-time data for paid tiers (uses authenticated SSE endpoint)
- [ ] Delayed data for free tier with upgrade prompt
- [ ] User's watchlist symbols highlighted

### AC-3: Plan badge
- [ ] Badge next to user name showing current plan (Free/Lite/Starter/Pro/Enterprise)
- [ ] Color: free=#71717A, lite=#3B82F6, starter=#F59E0B, pro=#8B5CF6, enterprise=#EF4444
- [ ] Click → /settings/billing

### AC-4: Tests
- [ ] Teste: Dashboard renders with sidebar
- [ ] Teste: Plan badge shows correct tier
- [ ] Teste: Free user sees delayed label
- [ ] Teste: Paid user sees real-time label
