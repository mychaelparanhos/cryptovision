# S2B-8: Referral Reward

**Epic:** Sprint 2 Expandida — Parte B (Ciclo de Venda)
**Status:** Draft
**Type:** Backend + Frontend
**Effort:** S (meio dia)
**Priority:** P2 (cherry-pick from CEO Plan)
**Depends on:** S2B-1 (auth with referral code capture), S2B-2 (Stripe)

## Story

Como usuário que referiu um amigo, quero receber 1 mês grátis quando meu referido fizer upgrade para plano pago, para ser incentivado a compartilhar o CryptoVision.

## Acceptance Criteria

### AC-1: Referral code generation
- [ ] Each user gets a unique referral code on account creation
- [ ] Code format: `CV-{first4chars-of-userId}` (human readable)
- [ ] Referral code stored in `users.referralCode` (or waitlist.referral_code if already exists)

### AC-2: Referral link sharing
- [ ] Dashboard Settings → "Share & Earn" section
- [ ] Shows referral link: `cryptovision.com/signup?ref=CV-XXXX`
- [ ] Copy button with "Copied!" feedback
- [ ] Shows referral stats: "X friends signed up, Y upgraded"

### AC-3: Referral reward
- [ ] When referred user upgrades to paid plan (checkout.session.completed)
- [ ] Referrer gets Stripe credit (1 month of their current plan value)
- [ ] Applied via Stripe customer balance adjustment
- [ ] Email notification to referrer: "You earned 1 month free!"

### AC-4: Tests
- [ ] Teste: Referral code generated on signup
- [ ] Teste: Referral link captures code on new signup
- [ ] Teste: Credit applied when referred user upgrades
- [ ] Teste: Referral stats endpoint returns correct counts
