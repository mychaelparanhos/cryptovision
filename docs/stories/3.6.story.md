## Story

Como novo usuário do CryptoVision que acabou de criar conta, quero ser guiado por um onboarding rápido, para entender o que o produto oferece e configurar meu perfil básico.

## Acceptance Criteria

### AC-1: Onboarding modal/wizard
- [ ] Appears on first login (onboardingCompleted === false)
- [ ] 3 steps: Welcome → Select interests (symbols) → Choose plan
- [ ] Step 1: Welcome message + product value prop
- [ ] Step 2: Select 3-5 symbols for watchlist (default: BTC, ETH, SOL)
- [ ] Step 3: Plan recommendation (Starter highlighted) with "Start Free" and "Upgrade" CTAs
- [ ] After completion: set onboardingCompleted=true, save watchlist

### AC-2: Skip option
- [ ] "Skip for now" button at every step
- [ ] Still marks onboarding as completed

### AC-3: Tests
- [ ] Teste: Onboarding shows on first login
- [ ] Teste: Onboarding doesn't show after completion
- [ ] Teste: Watchlist saved on completion
