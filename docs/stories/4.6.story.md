# S2C-6: Weekly Digest Email

**Epic:** Sprint 2 Expandida — Parte C (Entrega do Valor Pago)
**Status:** Draft
**Type:** Backend (Email + Cron)
**Effort:** S (half day)
**Priority:** P2 (engagement/retention)
**Depends on:** Resend package (already installed), User table (existing)

## Story

Como usuario do CryptoVision, quero receber um email semanal com os 5 maiores destaques do mercado (maiores liquidacoes, funding extremos, mudancas de OI), para manter-me informado mesmo sem acessar o dashboard.
