## Story

Como early adopter do CryptoVision, quero usar o código EARLYBIRD50 para obter 50% de desconto permanente, para ser recompensado por adotar o produto cedo.

## Acceptance Criteria

### AC-1: Stripe coupon
- [ ] Coupon EARLYBIRD50 criado no Stripe: 50% off forever, max 100 redemptions
- [ ] Coupon applied automatically when code entered on checkout

### AC-2: Coupon input on pricing page
- [ ] Input field "Have a coupon?" below pricing cards
- [ ] Validates coupon via Stripe API before checkout
- [ ] Shows discount preview: "EARLYBIRD50 — 50% off forever!"
- [ ] Shows remaining redemptions: "42 of 100 left"
- [ ] Error state: "Invalid coupon" or "Coupon expired"

### AC-3: Coupon counter on landing
- [ ] Landing page pricing section shows: "🔥 Early bird: 50% off — X spots left"
- [ ] Counter fetched from Stripe API (coupon.times_redeemed vs max_redemptions)
- [ ] Disappears when fully redeemed

### AC-4: Tests
- [ ] Teste: Coupon validation API returns valid/invalid
- [ ] Teste: Coupon applied to checkout session
- [ ] Teste: Counter shows remaining spots
