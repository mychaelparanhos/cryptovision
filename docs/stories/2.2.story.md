# S2A-2: Funding Rate Comparison Big 3

**Epic:** Sprint 2 Expandida — Parte A (Free Tier Robusto)
**Status:** Ready for Review
**Type:** Frontend + Backend (Full-Stack)
**Effort:** M (1 dia)
**Priority:** P1
**Depends on:** S2A-0 (publisher aggregation + Redis singleton)

## Story

Como trader Leo, quero ver uma tabela comparativa de funding rates entre Binance, Bybit e OKX lado a lado, para identificar divergências de funding que indicam possível squeeze ou oportunidade de arbitragem.

## Acceptance Criteria

### AC-1: Funding comparison table
- [ ] Componente `FundingComparison` exibe tabela com colunas: Symbol | Binance Rate | Bybit Rate | OKX Rate | Spread | Signal
- [ ] Dados de 20 símbolos (SUPPORTED_SYMBOLS) com funding rate de cada exchange
- [ ] Coluna "Spread" calcula max(rate) - min(rate) entre exchanges
- [ ] Coluna "Signal" mostra badge: "DIVERGENT" (#F59E0B amber) quando spread > 0.01% (configurável)
- [ ] Sort por: Symbol (default), Spread (desc), qualquer coluna
- [ ] Search/filter por symbol name
- [ ] Rates formatados com 4 decimais (ex: 0.0100%)

### AC-2: Visual indicators
- [ ] Cells com rate positivo: texto #22C55E (verde)
- [ ] Cells com rate negativo: texto #EF4444 (vermelho)
- [ ] Cells com rate zero/neutral: texto #A1A1AA (secondary)
- [ ] Exchange sem dados: cell mostra "—" em #71717A (muted)
- [ ] Row highlight on hover: #27272A background
- [ ] Education tooltip no header "Funding Rate": explica o que é e por que importa

### AC-3: Data source
- [ ] API endpoint: `GET /api/v1/public/funding/comparison`
- [ ] Lê `cv:cache:funding:{exchange}:{symbol}` para cada exchange via MGET
- [ ] Calcula spread e signal server-side
- [ ] Response inclui `predictedRate` e `nextFundingTime` por exchange
- [ ] Delayed 15min para free tier (handled by cache TTL)
- [ ] Auto-refresh a cada 60s

### AC-4: Landing page integration
- [ ] Componente integrado na landing como seção "Funding Rate Comparison"
- [ ] Posicionado abaixo do heatmap, acima do screener
- [ ] Mostra top 10 por spread (expandable para 20)
- [ ] CTA: "See all 20 pairs" expande a tabela

### AC-5: Testes
- [ ] Teste: FundingComparison renderiza com mock data (3 exchanges, 5 symbols)
- [ ] Teste: Spread calcula corretamente com rates diferentes
- [ ] Teste: Signal badge aparece quando spread > threshold
- [ ] Teste: Sort funciona por todas as colunas
- [ ] Teste: API endpoint retorna comparison data do cache
- [ ] Teste: API endpoint handles missing exchange data gracefully

- Table: border #27272A, padding 12px cells, JetBrains Mono para números

- Responsive: mobile shows 2 exchanges + spread, scroll para terceira
